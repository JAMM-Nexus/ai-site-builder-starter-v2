# 👋 Привіт! Я твій AI Site Builder

Я Claude — твій персональний AI-розробник у Claude Code Desktop. Готовий будувати **скільки завгодно** сайтів і проєктів за описом. Кожен проєкт живе у власній папці і деплоїться окремо.

## 0. Pre-flight (один раз перед стартом)

Перевір що `.env` у папці проєкту заповнений:

✅ `CLOUDFLARE_API_TOKEN` — з [dash.cloudflare.com → API Tokens](https://dash.cloudflare.com/profile/api-tokens)
✅ `CLOUDFLARE_ACCOUNT_ID` — з sidebar Workers & Pages

Це **глобальні** креди — заповнюються один раз, працюють для всіх твоїх сайтів.

Namecheap-змінні залишай порожніми — потрібні тільки для власного домену (`/domain`).

## Що я вмію

✅ Створити будь-який сайт або проєкт з опису
✅ Тримати **багато** сайтів паралельно у `projects/<slug>/`
✅ Адаптувати дизайн за референсом (фото, посилання)
✅ Опублікувати будь-який сайт на Cloudflare Pages за 30 секунд
✅ Редагувати елементи на льоту
✅ Перемикатись між сайтами командою `/use <slug>`

## Доступні команди

| Команда | Що робить |
|---------|-----------|
| `/status` | Перевірити креди + список твоїх проєктів |
| `/new-site <slug> [опис]` | Створити новий сайт у `projects/<slug>/` |
| `/edit [slug] [що змінити]` | Відредагувати сайт (активний якщо slug не вказаний) |
| `/deploy [slug]` | Опублікувати на Cloudflare Pages |
| `/list` | Показати всі мої сайти + їх URL |
| `/use <slug>` | Зробити сайт активним |
| `/domain [slug]` | Опціонально: підключити власний домен |

## Починаємо

Перш ніж будувати — перевір налаштування:
```
/status
```

Якщо все ✅ — кажи що будувати. **Slug** = коротке ім'я у kebab-case (`coffee-shop`, `portfolio`, `cafe-landing`):

```
/new-site cafe-landing Лендінг для кав'ярні, мінімалістичний стиль,
темні кольори, меню з 3 позиціями
```

**Приклади:**
- `/new-site portfolio Портфоліо фотографа, тепла палітра`
- `/new-site law-firm Бізнес-сайт юридичної фірми, корпоративний синій`
- `/new-site product-launch Сторінка запуску продукту, dark theme, один CTA`

## Другий, третій, сотий сайт

Просто створи ще один — попередні не зникають:
```
/new-site portfolio Портфоліо фотографа
/deploy
```

`/list` покаже всі. `/use <slug>` перемикає активний.
