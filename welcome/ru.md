# 👋 Привет! Я твой AI Site Builder

Я Claude — твой личный AI-разработчик в Claude Code Desktop. Готов строить **сколько угодно** сайтов и проектов по описанию. Каждый проект живёт в своей папке и деплоится отдельно.

## 0. Pre-flight (один раз перед стартом)

Проверь что `.env` в папке проекта заполнен:

✅ `CLOUDFLARE_API_TOKEN` — из [dash.cloudflare.com → API Tokens](https://dash.cloudflare.com/profile/api-tokens)
✅ `CLOUDFLARE_ACCOUNT_ID` — из sidebar Workers & Pages

Это **глобальные** креды — заполняются один раз, работают для всех твоих сайтов.

Namecheap-переменные оставь пустыми — нужны только для своего домена (`/domain`).

## Что я умею

✅ Создать любой сайт или проект по описанию
✅ Держать **много** сайтов параллельно в `projects/<slug>/`
✅ Адаптировать дизайн по референсу (фото, ссылка)
✅ Опубликовать любой сайт на Cloudflare Pages за 30 секунд
✅ Редактировать элементы на лету
✅ Переключаться между сайтами командой `/use <slug>`

## Доступные команды

| Команда | Что делает |
|---------|-----------|
| `/status` | Проверить креды + список проектов |
| `/new-site <slug> [описание]` | Создать новый сайт в `projects/<slug>/` |
| `/edit [slug] [что изменить]` | Отредактировать сайт (активный, если slug не указан) |
| `/deploy [slug]` | Опубликовать на Cloudflare Pages |
| `/list` | Показать все мои сайты + их URL |
| `/use <slug>` | Сделать сайт активным |
| `/domain [slug]` | Опционально: подключить свой домен |

## Начинаем

Прежде чем строить — проверь настройки:
```
/status
```

Если всё ✅ — говори что строить. **Slug** = короткое имя в kebab-case (`coffee-shop`, `portfolio`, `cafe-landing`):

```
/new-site cafe-landing Лендинг для кофейни, минималистичный стиль,
тёмные цвета, меню из 3 позиций
```

**Примеры:**
- `/new-site portfolio Портфолио фотографа, тёплая палитра`
- `/new-site law-firm Бизнес-сайт юридической фирмы, корпоративный синий`
- `/new-site product-launch Страница запуска продукта, dark theme, один CTA`

## Второй, третий, сотый сайт

Просто создай ещё один — предыдущие не пропадут:
```
/new-site portfolio Портфолио фотографа
/deploy
```

`/list` покажет все. `/use <slug>` переключает активный.
